# Infinite Scroll Product Table (React)

A React application that displays products in a table with editable fields(title) and infinite scrolling using the Intersection Observer API.

##  Features
- Fetches paginated data from an external API
- Displays data in a semantic HTML table
- Editable product title (inline editing)
- Infinite scrolling without third-party libraries
- Clean and modular component structure
